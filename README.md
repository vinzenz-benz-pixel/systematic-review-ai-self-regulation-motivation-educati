# systematic-review-ai-self-regulation-motivation-educati
Review Möglichkeiten KI in der Bildung im Bereich Förderung motivationale / metakognitive Selbstregulation. 

Thema:
Der Review soll Mechanismen im Einsatz von künstlicher Intelligenz zur Förderung von motivationaler und metakognitiver Selbstregulation untersuchen. Dabei soll es darum gehen, welche Strategien zur Förderung der Selbstregulation es gibt und wie das Potential von KI in diesem Feld aussieht. Wichtig ist, dass der Review kein starres Bild des Ist-Zustandes darstellt, sondern zukünftigen mehrwert liefert.

Ziel:
Überblick zu versuchen der Förderung von Selbstregulation durch KI.
Identifikation von Strategieansätzen für zukünftige Untersuchungen.
Die Rolle von Datenerhebungen und Datenanalyse in der Förderung von Selbstregulation durch KI.

Forschungsinteresse:
Förderung von Selbstregulation
Motivation
Datensammlung und Auswertung von Schülern
Lehrkräfteschulungen zur Nutzung von Schülerdaten in didaktischen, methodischen und pädagogischen Entscheidungen

Methodik:
Systematischer Review nach PriSMA
Living Review
